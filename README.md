Şifreleme (cipher, encryption)
- Açık metin bir anahtar ve algoritma ile kapalı metine dönüştürme.
  e,(x) = y

Şifre çözme (decipher, decryption)
- Kapalı metnin bir anahtar ve algoritma ile açık metine dönüştürmesi.
  d,(y) = x
